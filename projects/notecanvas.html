Overview

NoteCanvas is a powerful digital note-taking app optimized for the iPad and Apple Pencil. It combines the natural feeling of writing on paper with the power of digital organization and editing. The app features multiple paper styles, pen types, and organization tools to create a complete note-taking experience for students, professionals, and artists.

Key Features

  • Optimized for Apple Pencil with pressure sensitivity and tilt support
  • Multiple paper styles (lined, grid, dot grid, planner templates)
  • Variety of pen, marker, and brush styles with customizable colors
  • Handwriting recognition and search capabilities
  • Smart shape recognition for drawing perfect shapes
  • Folder organization system with tags and favorites
  • Automatic cloud backup and sync across devices
  • PDF import, annotation, and export

Technical Details

NoteCanvas was built using Swift with PencilKit for the drawing functionality, providing a smooth and responsive writing experience. The app uses CoreData for local storage and CloudKit for syncing notes across devices. Advanced features like handwriting recognition were implemented using Vision framework.

Swift PencilKit CloudKit CoreData Vision PDFKit

Development Challenges

One of the main challenges was creating a writing experience that felt natural and responsive, with minimal latency. This required careful optimization of the drawing engine and integration with PencilKit. Another challenge was implementing the handwriting recognition system that could accurately convert handwritten notes to searchable text across different handwriting styles.
