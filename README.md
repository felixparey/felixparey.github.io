# Felix Parey – Portfolio Website

My personal portfolio website showcasing iOS, iPadOS, and visionOS projects. This site serves as a project showcase and central hub for privacy policies and terms of use for my mobile applications.

## 🎯 Purpose

- **Project Showcase**: Display my mobile app development work
- **Legal Hub**: Centralized privacy policy and terms of use for my apps
- **Professional Presence**: Clean, modern portfolio representation

## 🗂️ Project Structure

```
├── index.html              # Main portfolio page
├── assets/                 # All media assets
│   ├── icons/             # App icons
│   ├── images/            # Project screenshots & images
│   └── profile/           # Profile/personal images
├── css/                   # Stylesheets
│   ├── styles.css         # Main styles
│   └── mobile.css         # Mobile responsiveness
├── js/                    # JavaScript files
│   └── script.js          # Main functionality
└── pages/                 # Additional pages
    ├── legal/             # Legal documents
    │   ├── privacy-policy.html
    │   ├── terms-of-use.html
    │   ├── legal-notice.html
    │   └── support-*.html
    └── projects/          # Project detail pages
        ├── basicstrategy.html
        ├── drawingcompetition.html
        └── duckrescue.html
```

## 🛠️ Technologies

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with glassmorphism design
- **Vanilla JavaScript** - Interactive functionality
- **Font Awesome** - Icons
- **GitHub Pages** - Static hosting

## 🚀 Features

- **Responsive Design**: Mobile-first approach with glassmorphism UI
- **Project Gallery**: Detailed showcase of mobile applications
- **Clean Navigation**: Smooth scrolling and intuitive user experience
- **Legal Compliance**: Integrated privacy policy and terms of use
- **No Tracking**: Privacy-focused, no analytics or third-party scripts
